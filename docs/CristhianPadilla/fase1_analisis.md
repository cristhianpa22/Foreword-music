Dark mode

Cómo se implementa Se implementa utilizando @custom-variant que define la condición de activación para las utilidades del prefijo dark:. La implementación usa la sintaxis CSS &:where(.dark, .dark *). -Estrategia por clase vs modo automático La estrategia por clase permite con un interruptor elegir entre tema claro y oscuro, anulando la preferencia del sistema operativo, y el modo automático solo respeta las preferencias del sistema operativo. La estrategia por clase permite recordar la elección del usuario en la página web; el modo automático no lo puede hacer. Se puede controlar de forma síncrona en el mediante JavaScript para prevenir el parpadeo blanco; el modo automático no, haciendo que cuando se recarga la página tenga el riesgo de que se produzca el parpadeo blanco.
